机器学习（5）训练集和测试集

发表于 2020-04-20 分类于 机器学习

拆分数据集

如果我们只有一个数据集，我们可以将这个大型数据集分成两个小数据集： 一个用于训练，一个用于测试。 这两个数据集需要相互独立。另外，我们需要先进行随机化，再拆分数据

测试集的要求

1）规模足够大，可产生具有统计意义的结果。
2）能代表整个数据集。 也就是说，挑选的测试集的特征应该与训练集的特征相同。

训练集和测试集规模

训练集规模越大，模型的学习效果就越好。 测试集规模越大，我们对于评估指标的信心越充足

一般来说，训练集数据的数量应占2/3到4/5。 小数据量时代，数据几十万以下时， 通常将数据集的80%作为训练集，20%作为测试集
如果我们的数据集规模非常大， 那么应该把划分的比例缩小。 比如百万数据量时测试集只需要占比1%就可以

注意：绝对不要对测试集进行训练

机器学习（4）过拟合和泛化

发表于 2020-04-20 分类于 机器学习

1. 过拟合和泛化

泛化是指模型很好地拟合以前未见过的新数据的能力
过拟合指模型最初可能过分符合训练数据， 而无法很好地对之前未出现的新数据进行分类，泛化能力差

如果模型过于复杂， 那么过拟合真的会成为问题

模型是否出色的标准

14世纪的奥卡姆的威廉，提出了“奥卡姆剃刀”这一概念， 大意是说模型应尽可能简单

我们从该分布抽取一批数据， 对这些数据进行训练， 这就是训练集

我们再从该分布另外抽取一批数据， 用于测试模型称之为测试集， 一般来说， 在测试集上表现是否良好，是衡量能否在新数据上表现良好的有用指标

2. 机器学习细则

这里有些注意事项， 也就是机器学习的一些细则

1）我们要以独立且一致的方式从该分布抽取样本。 样本之间不会互相影响。 我们不以任何方式主动产生偏差
2）分布是平稳的，不会随时间发生变化。 即分布在数据集内不会发生变化
3）我们始终从同一个分布提取样本， 不会突然开始从其他分布提取样本

以上是非常关键的假设， 我们在监督式机器学习中 所做的许多工作都是以这些假设为基础

机器学习（3）降低损失

发表于 2020-04-14 更新于 2020-04-20 分类于 机器学习

好的模型处于损失最低时，那么我们就要力求损失函数最小，降低损失。如果我们能够确定损失函数要往参数空间的哪个方向前进就再好不过了。 沿着这个方向， 我们可以知道所选择的每组新的超参数 都比之前一组参数的损失小一些。 要获得具体方向， 一种方法是计算梯度， 即与模型参数相关的损失函数的导数
我们在能够尽可能降低损失的方向上反复采取小步。 我们将这些小步称为梯度步长 （但它们实际上是负梯度步长） 这种优化策略称为梯度下降法 这样我们就能高效地更新模型参数。 可以将其视为一种迭代法，目的是找到损失函数的最小值

1. 迭代方法

“模型”部分将一个或多个特征作为输入， 然后返回一个预测作为输出

“计算损失”部分是模型将要使用的损失函数， 计算预测标签和真实标签的差距

“计算参数更新”部分， 机器学习系统就是在此部分检查损失函数的值， 并为模型的权重和偏置生成新值。 然后再重新评估， 这种学习过程会持续迭代。 直到总体损失不再变化或至少变化极其缓慢为止， 这时候，我们可以说该模型已收敛

梯度下降法

我们随机选择一个初始值， 就可以看到对应的损失。 接着，我们可以计算负梯度， 从而得知往哪个方向前进， 才能尽可能减小损失。

如果我们朝着这个方向沿着梯度前进， 就会看到新的损失。 可以继续 朝着这个方向沿着梯度前进， 直到超过局部最小值。

而负梯度会指示我们沿着来时的方向返回。 在负梯度的方向上， 我们应该迈着多大的步伐前进？

学习率（步长）

学习率也就是沿负梯度方向迈步的大小，可自行调节

如果学习速率非常小， 那么我们的梯度步伐就小而密。 那么这需要进行大量的计算，才能到达最小值，就会花费太长的学习时间
如果学习速率非常大， 那么我们会在负梯度方向迈出一大步，有可能超过局部最小值，损失甚至会比之前还大。在这种情况下，应该尝试将学习速率减慢，差不多减慢一个数量级

权重初始化

神经网络的形状不像碗， 而更像是蛋架。 有很多可能的最小值， 其中一些最小值要优于其他最小值

这样看来，初始值确实很重要。 我们之后会进一步说明

随机梯度下降法和小批量梯度下降法

我们先花点时间思考一下效率。 在计算损失函数的梯度时， 我们通过学过的数学知道， 应该计算数据集中所有样本的梯度。 只有这样才能确保我们的梯度步伐 正好朝着正确的方向

对于样本数量， 达到百万或十亿的大型数据集。 要迈出每一步， 都需要进行大量的计算。 根据经验，人们发现， 无需使用整个数据集， 只要计算单个样本的损失函数， 那么大部分也都是适用的。 尽管这样需要迈出更多整体步伐， 但为获得最佳解决方案 而需要进行的总计算量往往也少得多。 这称为随机梯度下降法

在实际运用中， 我们采用折中的解决方案。 我们并非使用 一个样本或整个数据集， 而是使用一小部分样本， 差不多10-1000个样本。 这称为小批量梯度下降法

机器学习（2）回归模型与损失

发表于 2020-04-14 分类于 机器学习

1.回归模型

机器学习中的模型就是我们忙活半天想要得到的产物，方程如下：

$$\hat{y} = b + w_1x_1 + w_2x_2 + ……$$

\(\hat{y}\)：预测标签（理想输出值）
b ：偏差，有时用$w_0$表示
\( w_1 \)：特征1的权重
\(x_1\)：特征1（一直输入项）
……
刚开始模型一片空白，参数w和b都是待定的，通过训练来得到，那什么又是训练呢？

2.训练与损失

训练模型表示通过有标签样本来学习所有权重和偏差的理想值
损失是对糟糕预测的惩罚。 也就是说，损失是一个数值， 表示对于单个样本而言模型预测的准确程度
训练模型的目标是从所有样本中找到一组平均损失“较小”的权重和偏差，此时的模型就是我们想要的，性能最好的模型

损失函数

最常见的损失函数有如下几种：
1) 平方损失（L2损失）
单个样本的平方损失， = 预测值和标签值之差的平方 = (观察值 - 预测值)的平方 = (y - \(\hat{y}\))的平方
2）均方误差（MSE）
指的是每个样本的平均平方损失。 求出各个样本的所有平方损失之和， 然后除以样本数量
$$MSE = \frac{1}{N} \sum_{(x,y)\in D} (y - \hat{y}）^2$$
x：预测时使用的已知特征集
y：样本的标签
\(\hat{y}\)：模型预测出的标签，\(\hat{y}\) = prediction(x)

1.简介

机器学习，是什么呢？用我自己的理解，就是一个模型，它能够吃进一大坨数据对（x,y)，经过消化，找到x和y的内在联系，并生成一个模型。利用这个模型，你再给它新的x，他就能找到对应的y。简单来说，就是因果归纳机。

传统编程与机器学习


举例来说，传统编程是已知x和方程 y = 2x + 1（方程既程序），求y的值。但对于机器学习，我知道一堆x和y的值对，要求一个x和y的对应关系。

2.基本概念

基本概念

1）特征：使我们表示数据的方式，是输入变量x
2）标签：标签是指我们要预测的真实事物y
3）样本：有标签的样本{特征，标签}，用于训练模型
无标签的样本{特征，？}，用于对新数据作出预测
4）模型：定义了特征与标签之间的关系，是执行预测的工具 (x —> model —> y)

模型生命周期的两个重要阶段：训练和推断
训练是向模型输入有标签的样本，让模型学习特征与标签之间的关系
推断是将训练后的模型用于无标签样本，作出预测

你带小朋友去野生动物园去看马，在马圈旁边，你指着一匹说这是马，另一匹也是马，这都是马。过一会旁边遛过来一头鹿，你说，这不是马。听着你不断唠叨，小朋友就会产生认知模式，学会分辨马这种动物。这个学习的过程就是训练，学会的认知模式就是模型。训练过后，再跑来一个动物，你让小朋友判断是不是马，这就是预测。如果你的模型够好，就不会发生指鹿为马的事情。
