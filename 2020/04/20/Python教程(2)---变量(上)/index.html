变量命名规范

  • 标识符的第一个字符必须是字母或下划线
  • 标识符名称的其他部分可以由字母、下划线、或数字组成
  • 标识符是对大小写敏感的（a和A是不一样的变量名）

变量类型

1）数值类型

变量为数字，最常见的一种，没啥好说
列举一些函数和运算：
number = 10
number = number + 10
number += 10#在自己基础上加十
number /= 10

min(10, 23, 76, 5)#返回最小值
max(10, 23, 76, 5)

divmod(10, 3) #返回10除以3的商和余数

math.pow(3, 10) #3的十次方
math.radians(180) #角度转弧度

2） bool类型

有两个值：True， False两种
True ==1, False == 0

bool运算：

与运算：True and False (一假即假)
或运算：True or False （一真即真）
非运算：not True

有以下操作符会返回bool值：

操作符 解释
< 小于
<= 小于等于
> 大于
>= 大于等于
== 等于
!= 不等于
is 是相同对象

3）字符串类型

1>字符串运算:

  • 加法操作

    把两个字符串拼接

  • 乘法操作

    打印5遍

  • len(line)
    查看字符串长度

  • id(line)
    查看字符串在内存中的地址

2>切片操作

line = 'hao hao xue xi tian tian xiang shang'

print(line[ :10]) #取前十个字符

print(0:15:2)#前十个字符，隔一个取一个

print(line[-10: ]) #取后十个字符
print(line[5])#取单字符

字符串的index从0开始编。
