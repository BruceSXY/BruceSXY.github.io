因此函数也可作为另一个函数的参数，既函数套函数
了解这点以后，我们来介绍装饰器
装饰器是一个函数，让我们套在其他函数上，来修饰增强其他函数，但调用时还是只调用那个其他函数即可
我们来看一个例子，执行一个函数以后，写一条日志

这样做逻辑上是没问题的，功能是实现了，但是我们调用的时候不再是调用真正的业务逻辑 test 函数，而是换成了 use_logging 函数，这就破坏了原有的代码结构，我们换另一种写法

use_logging 就是一个装饰器，它一个普通的函数，它把执行真正业务逻辑的函数 func 包裹在其中，看起来像 test 被 use_logging 装饰了一样，use_logging 返回的也是一个函数，这个函数的名字叫 wrapper.

符号@就是装饰器的语法糖，它放在函数开始定义的地方，这样就可以省略最后一步再次赋值的操作

有了 @ ，我们就可以省去test = use_logging(test)这一句了，直接调用 test( ) 即可得到想要的结果
